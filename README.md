Put 20-lubuntu.conf into /etc/lightdm/lightdm.conf.d

Put 95-touchpad_toggle.rules into /etc/udev/rules.d

tptoggle.sh & wifitoggle.sh are to be used in ~/.config/openbox/lubuntu-rc.xml
